# 说明

基于[Jekyll](https://jekyllrb.com/)搭建的静态博客

# 安装

1. 下载并安装Ruby环境[https://www.ruby-lang.org/en/downloads/](https://www.ruby-lang.org/en/downloads/)
2. 安装Jekyll
    ```
    gem install jekyll bundle
    ``` 
3. 安装项目依赖
    ```
    bundle install
    ```
4. 启动
    ```
    bundle exec jekyll server 
    ```
5. 打包
    ```
    bundle exec jekyll build
    ```
